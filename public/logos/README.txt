LOGO FILES DIRECTORY
====================

Please save the Arbutus logo files to this directory:

1. arbutus-logo-full.png  (Full logo with text "ARBUTUS - Drive Your Analytics")
2. arbutus-icon.png       (Circular orange icon only)

The login page has been designed to use the Arbutus brand colors:
- Primary Orange: #E85D1C
- Darker Orange: #C74915

The current implementation uses CSS-based logo placeholders that match the brand.
If you add the actual PNG files here, you can update the login page to use Next.js Image components.
